## Лабораторная работа №3. Работа с Markdown-разметкой и базовое использование LaTeX в документации проекта

#### Markdown — это легкий язык разметки, который позволяет делать текст структурированным и красивым с помощью простых символов и команд. Он был создан для того, чтобы писать отформатированный текст легко и быстро, без необходимости использовать сложные редакторы или HTML
#### Где он применяется:

Markdown широко используется для написания документации, README-файлов на GitHub, блогов, заметок и даже в системах управления контентом. Благодаря своей простоте его любят использовать разработчики, техспециалисты и все, кому нужно создавать структурированный текст без особых усилий.

#### Разработчики используют Markdown, чтобы быстро документировать свой код, создавать инструкции, отчеты или комментарии, которые легко читаются и редактируются. Это помогает держать документацию в актуальном виде, а также делит работу над проектом более организованно. Markdown облегчает совместную работу, делая документацию понятной и приятной для чтения.
